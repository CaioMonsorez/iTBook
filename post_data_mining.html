Mineração de dados - Aprenda o que é, pra que serve e como iniciar o processo de Data Mining

Lista das bibliotecas e comandos mais utilizadas para Data Mining em Python e R

Posted by Caio Monsorez on December 11, 2020

O que é e pra que serve Data Mining?

É impossível entender o termo "Data Mining" se nao explicar o conceito de Dados e Mining separadamente. No mundo dos Cientistas de Dados, Data, ou dado é um conjunto de informaçoōes contidas em um arquivo de extensão json, csv, xml, por exemplo. Esses dados, se analisados, garimpados e limpados, pode-se obter previsōes de alguns fatos.

Por exemplo:

Previsao do climática. As previsões climáticas procuram nos dizer como será o clima daqui a alguns dias, se o planeta será mais quente ou mais frio daqui a alguns anos, se as chuvas serão mais frequentes ou menos intensas etc. Para sabermos se amanhã vai chover ou não, para prever um clima ensolarado ou nublado é necessário um conjunto de dados (caracteristicas) para chegar a conclusão da meterológica. Alguns dos numerosos dados que podem ser analisados pelos especialistas são normalmente condicionais:

- Nuvens mais negras que o habitual ( Sim ou nao?)
- Nuvens altas (sim ou nao?)
- Temperatura e umidade (sim ou nao?)
- Vento (sim ou nao?)
- A cor do céu (sim ou nao)

A partir desses dados respondidos, poderemos minerar os dados, ou seja explorar as informaçoōes para prever o clima. Quem já se está acostumado com o Excel, explora dados pra conseguir um resultado padrao pela tabela dinamica, por exemplo. A tabela dinamica reune todas as informacoes de maneira visualmente mais organizada. A mineração de dados serve ao objetivo principal de descobrir padrões entre grandes volumes de dados e transformá-los em informações mais refinadas.

Como minerar os dados?

Para melhor experiência de visualização de dados, a ferramenta que escolhi para produzir os codigos foi o Anaconda Navigator.

Existem inumeras ferramentas de codigo aberto pra minerar dados, como Rapid Miner, Orange, Weka, Knime, Apache Mahout, Rattle, Data Melt , inclusive para quem gosta de VSCode, também é possível e excelente minerar por lá. Mas, vou focar no Anaconda Navigator que é a ferramenta mais popular e amigável para Data Science.

O Anaconda Navigator contem centenas de pacotes instalados e podemos utilizar o python e R juntos. Dentro do Anaconda, utilizaremos a Aplicação cliente-servidor JUPYTER, que permite a edição e execução de notebooks via browser. Notebooks são documentos que contém código e elementos visuais como imagens, links, equações. A principal vantagema utilização de notebooks é para a descrição de análises e seus resultados de forma dinâmica e interativa.

Caso não haja a biblioteca instalada, é so utilizar o comando conda (gerenciamento de pacotes do Anaconda) para instalar pacotes novos.

Anaconda tem versões para Windows, Linux e OSX, e já vem com o Python instalado, apenas escolha qual a versão você quer trabalhar. Para fazer download do Anaconda, clique aqui.

Como Minerar Dados com Python?

Python sem duvidas, é a ferramenta mais querida pelos Cientistas de Dados pois possui uma rica biblioteca de análise exploratória preditiva e visual.

Para importar essas bibliotecas, utilize o comando import (nome da biblioteca) as (nome de como vc quer chamar sua lib)

Mas qual lib (biblioteca) mais usada pra minerar?

1. NumPy: Biblioteca Python para computação científica. Implementa arrays multidimensionais e permite a fácil execução de operações matemáticas e lógicas como ordenação, seleção, transformações, operações estatísticas básicas etc.

2. Matplotlib: Biblioteca Python 2D para a visualização e plotagem de gráficos. Pode ser utilizada para gerar diversos tipos de gráficos como histogramas, gráficos de barras, gráficos de pizza tudo de forma fácil e rápida.

3. Pandas: Esta biblioteca talvez seja a mais utilizada para análise de dados. Ela fornece ferramentas para manipulação de estruturas de dados de forma extremamente simples. Operações complexas que trabalham com matrizes e vetores podem ser facilmente realizadas com uma ótima performance.

4. Scikit-Learn: Biblioteca Python para trabalhar com Machine Learn (Aprendizado de Máquina). Contém diversos algoritmos implementados, métodos de análise e processamento de dados, métricas de avaliação etc. Essa é uma biblioteca extremamente útil para o cientista de dados.

5. NLTK: é uma plataforma líder para a construção de programas Python para trabalhar com dados de linguagem humana. Ele fornece interfaces fáceis de usar para mais de 50 corpora e recursos lexicais como o WordNet, juntamente com um conjunto de bibliotecas de processamento de texto para classificação, tokenização, stemming, tagging, análise e raciocínio semântico

6. Scrapy: Biblioteca Python para a raspagem ou coleta de dados a partir da Web. É possível coletar dados de sites, redes sociais, fóruns e diversos outros canais utilizando uma linguagem simples e objetiva. Extremamente útil para a geração de bases de dados.

Como Minerar Dados com R?

O R é uma ferramenta destinada ao desenvolvimento de sistemas de apoio à decisão de análise de dados, bem como à execução de tarefas mais complexas que envolvam programação. Ao mesmo tempo, é uma linguagem de programação e um ambiente para computação estatística e gráficos.

Mas qual lib (biblioteca) mais usada pra Data Mining em R?

1. Rpart: pode ser utilizada tanto para árvores de regressão como de classificação.

2. lm: Bpode ser usada para obter regressões lineares

3. nnet: é utilizada no âmbito das redes neuronais
